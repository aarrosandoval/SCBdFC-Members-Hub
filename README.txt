# SCBdFC Hub Setup (Flat Version)

1. Upload ALL these files directly into the **root of your GitHub repo** (no folders needed).

2. Go to repo Settings ▸ Pages ▸ Source = Deploy from branch ▸ main branch, /root.

3. Wait 1–2 minutes, then visit: https://<your-username>.github.io/<repo-name>/

4. Test:
   - index.html loads styled homepage
   - events.html shows calendar (with FullCalendar CDN)
   - results.html shows sample results

5. To update events/deadlines/results, just edit the JSON files directly in your repo.
